# Premium Music System

Experience high-quality, lag-free music playback powered by Lavalink and the Riffy music engine.

## Overview

Vermeil's music system is built for performance and sound quality. It supports a wide range of platforms, including YouTube, Spotify, and SoundCloud. With built-in audio filters and 24/7 mode, you can turn your Discord server into a professional audio lounge.

### Features
- **High Fidelity**: Powered by Lavalink for crystal-clear audio.
- **Audio Filters**: Bassboost, Nightcore, Vaporwave, and more.
- **Auto-Play**: Automatically finds and plays related songs when the queue ends.
- **24/7 Mode**: Keep the bot in the voice channel even when it's empty.

## Setup

1. Join a voice channel in your server.
2. Use `!play <song name or URL>` to start listening.
3. (Optional) Use `!247` to keep the bot in the channel permanently (Admin only).

## Commands

### 🎵 Playback & Queue
| Command | Description |
|---------|-------------|
| `play` | Play a song or add a URL to the queue. |
| `skip` | Skip the current track. |
| `pause` | Pause the current playback. |
| `resume` | Resume paused playback. |
| `stop` | Stop the music and clear the entire queue. |
| `nowplaying` | Show details about the song currently playing. |
| `queue` | View the upcoming songs in the queue. |
| `shuffle` | Randomize the order of the queue. |
| `loop` | Toggle looping for the current track or the entire queue. |

### 🛠️ Management & Controls
| Command | Description |
|---------|-------------|
| `volume` | Adjust the player volume (1-200%). |
| `seek` | Jump to a specific timestamp in the song. |
| `move` | Change the position of a track within the queue. |
| `remove` | Remove a specific track from the queue by its index. |
| `247` | Toggle 24/7 mode to keep the bot in the voice channel. |
| `autoplay` | Automatically add related songs to the queue. |

### 🔊 Audio Filters
Use the `!filters` command to apply real-time audio effects to your music:
- **Bassboost**: Enhance the low-end frequencies.
- **Nightcore**: Increase speed and pitch for a high-energy vibe.
- **Vaporwave**: Slow down and pitch down for a relaxed, lo-fi feel.
- **Clear**: Remove all active filters.

## Supported Platforms
- **YouTube**: Search by title or paste a direct video/playlist link.
- **Spotify**: Paste any Spotify track, album, or playlist URL.
- **SoundCloud**: Search or paste direct SoundCloud links.
