# Information Commands

Vermeil provides a suite of information commands to help you keep track of your bot, server, and community statistics.

## Overview

Information commands are designed to give you a clear and detailed view of any aspect of your server. Whether you're checking server stats, user details, or the bot's health, these commands provide all the data you need.

### Key Features
- **Server Stats**: Get a complete overview of your server's health and growth.
- **User Details**: See when a user joined, their roles, and other important data.
- **Bot Health**: Monitor Vermeil's performance and uptime.
- **Support Links**: Get the official invite and support server links.

## Commands

### 🔍 Bot & Server Info
| Command | Description |
|---------|-------------|
| `botinfo` | View detailed information about the bot, including version and uptime. |
| `serverinfo` | View comprehensive statistics about the current server. |
| `invite` | Get the official bot invite link to add Vermeil to other servers. |
| `supportserver` | Get an invite link to the official Vermeil support community. |

### 👤 User Information
| Command | Description |
|---------|-------------|
| `userinfo` | View detailed information about a specific member or yourself. |

## Server Information
The `!serverinfo` command provides a snapshot of your server, including:
- **Server Creation Date**: When the server was first created.
- **Member Breakdown**: Total members, bots, and online status.
- **Channel Counts**: Number of text, voice, and category channels.
- **Role List**: A list of all roles currently in the server.
- **Server Level**: If applicable, shows the server's boost level and status.

## User Information
Use `!userinfo` to learn more about anyone in your server. This includes their join dates (both for Discord and the server), their current roles, and their unique user ID. This is particularly useful for staff when identifying and managing community members.
