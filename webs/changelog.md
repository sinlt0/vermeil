# Changelog

All notable changes to the **Vermeil** project will be documented in this file.

## [1.0.0] - 2026-04-16

### Added
- **Fun Category:** Added Akinator (`aki`), random memes, jokes, 8ball, and cat/dog picture commands.
- **Anti-Nuke System:** Panic mode, whitelisting, and quarantining features.
- **Economy & RPG:** Deep economy system with battles, hunts, and jobs.
- **Premium Music:** High-quality audio powered by Lavalink with filters.
- **Giveaway Suite:** Advanced giveaways with role and level requirements.
- **Web Dashboard:** Initial framework for server management (beta).
- **Documentation:** Full documentation for all command categories.
- **Privacy & TOS:** Added legal pages for compliance.

### Improved
- **Autoplay System:** Completely rebuilt the autoplay logic to reliably play related tracks when the queue ends.
- **Website Design:** Modernized UI with glassmorphism and mobile responsiveness.
- **Navigation:** Added mobile hamburger menu and sticky documentation sidebar.
- **Performance:** Optimized command loading and database queries.

### Fixed
- **Mobile Layout:** Fixed navigation issues on small screens.
- **Docs Layout:** Fixed sidebar and content alignment on laptops.
- **Help Command:** Corrected category emojis and hidden categories logic.
